Una persona desde su ordenador tiene que crear el servidor ejecutando server.py y luego el resto (puede ser también la persona que creó el servidor, pero sin cerrar el servidor ya que se apagaría) se une.
Siempre tiene que haber un servidor abierto. Antes de crearse el servidor, te pide que pongas un nombre para el servidor.
Los clientes también eligen su nombre antes de conectarse al servidor y ellos son los que escriben mensajes y todo.
Las IPs de todo el mundo las puede ver el servidor.
Si el del servidor cierra el programa, se apaga todo.
